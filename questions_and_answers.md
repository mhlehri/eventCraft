###### 3. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
const food = ["🍕", "🍫", "🥑", "🍔"];
const info = { favoriteFood: food[0] };

info.favoriteFood = "🍝";

console.log(food);
```

- A: `['🍕', '🍫', '🥑', '🍔']`
- B: `['🍝', '🍫', '🥑', '🍔']`
- C: `['🍝', '🍕', '🍫', '🥑', '🍔']`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A

<i>food array, containing emoji elements like "🍕", "🍫", "🥑", "🍔" remains unchanged even after modifying the info.favoriteFood property to "🍝." The console output will still display the original food array as ["🍕", "🍫", "🥑", "🍔"].</i>

</p>
</details>
